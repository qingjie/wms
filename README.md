# WMS
基于SSM框架的仓库管理系统

在https://github.com/KEN-LJQ/WMS基础上，做了简单优化：
   1、优化了验证码，只使用数据
   2、优化了autocomplete（输入框输入空格，显示全部记录）
   3、优化了出入库的时候，详情展示（使用table进行展现）
   4、优化了登陆密码，新建用户默认密码为123456
